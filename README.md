# mapins

Project to get familiar with:

* Vagrant, puphpet
* Node.js
* Backbone
* MongoDB

## Setup

1. Add to /etc/hosts:

    ```
    192.168.56.101  local.mapins.com
    ```

2. Install dependencies. Maybe sudo needed

    ```
    npm install
    ```

3. Start VM

    ```
    cd vagrant/local
    vagrant up
    ```
